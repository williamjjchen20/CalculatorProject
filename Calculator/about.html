Useful Tips
1. If your answer is invalid, use the clear button to re-enter your expression.
2. There are no parentheses in this version, so PEMDAS is calculated without the first step. If there is a case where parentheses are needed, such as 6^(1/2), calculate the expression in the () first and use paste to place it in where it is needed.
3. Logarithm is only base 10, so if you need to calculate with another base, use the change of base formula. log base a of b = log(b)/log(a).
4. There is a bug for any operator that requires the Shift key. If you release shift too quickly, the button will appear permanently down. To fix, hold down shift for a longer duration and press the key that is locked once more. Clear the calculation line afterwards.
5. If you use GCD and LCM correctly, it will automatically change to the form gcd(a, b) or lcm(a, b). If you do not see it in this form on the calculation line, clear and re-enter.
